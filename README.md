# 🐲 QRモンスターファーム

世の中のQRコードからモンスターを召喚して育て、大会ランクSの頂点を目指すブラウザゲーム。

**▶ 遊ぶ:** https://isseioku09.github.io/monster-farm/

## 特徴

- 📷 QRコードをスキャンすると、そのQR固有のモンスターが召喚される（同じQR＝同じモンスター）
- 🕹️ 全151種族のドット絵モンスター（★4以上限定の伝説種18種を含む）
- 🌟 QRごとに隠された「御利益タイム」（曜日×時間）に読み込むと覚醒版が出現
- 🏋️ 週単位の育成（トレーニング無料・お金がなくても遊べる設計）
- 🏆 E→Sランクの公式大会
- ⚗️ 2体を掛け合わせる融合召喚（合成種）
- 🤝 モンスターをQR化して友だちと対戦／1台で2人対戦
- 📖 全151種の図鑑コンプリート要素

## 動かし方

ブラウザで上のURLを開くだけ。インストール・登録不要。セーブはブラウザ内（localStorage）。

カメラでのQRスキャンにはHTTPS環境が必要です（GitHub Pages上では動作します）。
